# BlackJack Vite

Pasos para ejecutar proyecto:

1. Clonar repositorio
2. Ejecutar ```npm install``` para recontruir los modulos de node
3. Correr el devServer ```npm run dev```
4. Abrir proyecto en: ```http://localhost:5174```
